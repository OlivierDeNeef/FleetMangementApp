---------------------------------------Database------------------------------------------------------------------------
1. Run het scrip "DatabaseSetup.sql" dat zich bevindt in het project DataAccesLayer in de map Sql
2. Volg te stappen om met de databank te verbinden in VS.
3. Run het script "TableScript.sql om te tabellen aan te maken.
4. Run het script "DummyData.sql" om de eerste data in te laden.


-------------------------------------Startup------------------------------------------------------------------------
1. Zorg ervoor dat het project "FleetManagermentApp" als "startup Project" is ingesteld. Dit kan met een rechtermuisklik op de titel van het project.
2. Druk op de knop met de groene pijl bovenaan midden van het scherm.

-------------------------------------Gebruik------------------------------------------------------------------------
1. Het programma wordt opgestart in de "Main Window". Dit is een overzicht met tabs bovenaan die naar verschillende delen van het programma leiden.
1b. Het sluiten van een window, reset de mogelijke ingevulde gegens.

Een Bestuurder heeft een Voertuig, Rijbewijs.

----TabBestuurders
2. De eerste tab geeft een overzicht van de Bestuurders in het programma. 
3. Vul een aantal velden in om te filteren op de gegevens van de bestuurder die je wilt vinden door een klik op "Zoeken".
4. Met de knop "nieuw", kan u een volledig nieuwe Bestuurder aanmaken.
5. Eenmaal er een resultaat in de grootste onderste tabel, kan u deze aanklikken.
6. Als er een resulaat geselecteerd is, komen de knoppen "Details", "Aanpassen" en "(De)Archiveren" actief.
7. De knop "Details" toont een nieuwe window met een overzicht van alle informatie over dit Voertuig, samen met de Bestuurder en Tankkaart
8. De knop "Aanpassen" toont een nieuwe window waar u de gegevens van de bestuurder kunt aanpassen.
9. De knop "DeAchiveren" zet de Bestuurder met een status op "Gearchiveerd". dit is bij Details ook te zien.
